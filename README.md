# BadGirlChat

BadGirlChat ist eine einfache, unzensierte Chat-App mit Tab-Navigation für Chat, Einstellungen, Erinnerungen und Fotos.

## Installation

```bash
npm install
```

## Start

```bash
npm start
```

Anschließend ist die App unter [http://localhost:3000](http://localhost:3000) erreichbar.
